My Projects

Below are some of my projects I've created over the years. Some of them have download links so you can test them out yourself!

🎮 Games

  • Galatigoo ⬅️ ➡️

    A 2D top-down tower defense game where you battle an enemy that can only be delayed, not defeated. Features a unique resource management system and endless gameplay strategies. I'm currently rewriting this game in my own game engine, stay tuned! In the meantime, download a demo of the game made in Unity below! Make sure to let me know what you think of the idea :]

    ⬇️ Download
  • Five Nights at Jacobsons ⬅️ ➡️

    My junior-year capstone project that I worked on with one other person who assisted with art. A satirical game which transforms teachers from my school into killer animatronics akin to those in the popular game "Five Nights at Freddy's". I can proudly say this game was present on every student's computer monitor until the end of the year (can't say the teachers were as happy).

    ⬇️ Download
  • Connect-4 (Rust)

    A very small terminal-based game I created to experiment with the programming language Rust. Features an AI opponent with customizable difficulty and a local PvP mode where you can play against another person.

    ⬇️ Download
  • Tic-Tac-Toe (C++)

    Another small terminal based game I created to experiment with minimax algorithms for video game AI; also just an excuse for me to dabble in C++ for a little bit. Only features playing against an AI opponent! Can you beat the impossible difficulty? (I think not)

    ⬇️ Download

🎵 Music

  • Funky Game Jam Music

    Super Funky Menu

    Super Funky Track

    These are two retro-pixel-style songs, ten seconds each, that I created for a Game Jam hosted in our school district. The game I created placed second place by only a few points, but, I still had fun making it! :]

  • Happy Chiptune

    Happi

    A very upbeat and cheery chiptune song! Definitely one of the more complicated works; I tried to emulate the visiting of multiple level biomes, like how you would in Super Mario Brothers. It came out really well!

  • Five Nights at Jacobsons

    First Menu Theme

    Second Menu Theme

    So, interesting story for these two menu tracks that I made for Five Nights at Jacobson (FNAJ). The first menu theme was created with an interesting synth I made by complete accident in FL Studio, but hey, it sounded cool!

    The second menu theme was a complete overhaul of the first theme, since I thought the game got scarier as we went further into the development process, but everyone seemed to love the first theme more than the second! Though, I guess the second is less of a theme and more so ambiance? Maybe less actually is more? 🤔

    Fun fact: some of the high-pitched chimes in the second theme is a heavily distorted recording of me hitting a nail with a hammer in my garage.

  • Scrapped Balloon Game

    Through the Wind

    A very calming (or at least it tries to be) classical piece that was intended to be used for a game project where you free roam the world in a massive hot air balloon. Unfortunately, the project as of now has been scrapped since I lost interest, but I might return to it some day!

  • Untitled Electronic

    Untitled

    An upbeat electronic track I created to practice mixing and propper panning with instruments. Also a sneaky bit of keyboard thrown in there because you can never go wrong with that. I actually really enjoyed how it turned out, despite being relatively basic. Oh also it's unnamed, so name it whatever you think fits I guess.

  • Mystery Track

    Detective

    A very L.A. Noire inspired piano swing piece. Probably one of the most complicated lead melody parts I've composed to this date. I feel like it's a little bumpy at some points, but my excuse is Jazz: the genre where mistakes and unintentional rhythms are APART of the composition!
